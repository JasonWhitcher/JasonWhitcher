Jasper Experience is a website that allows user's to list activities or events in Jasper County. The local population can view the site and know what there is to do in the area.

The site is built with Wordpress. It is centered around a custom plugin that uses a custom post type for the activity listings. Custom taxonomy is used to replace the default post categories. It also uses custom template for archive and single pages in the chosen theme.

The back end admin is customized for the site administrator.
